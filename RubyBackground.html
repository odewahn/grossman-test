Ruby Background

Ruby is a programming language that was invented in the mid-1990s by Yukihiro "Matz" Matsumoto in Japan.  The language was built to enable rapid development of programs and over time it has become increasingly popular.  Popular companies like Twitter use it and many websites have software that runs Ruby to make their websites work. 

We think it's a great teaching language as well. It is object-oriented from its foundation which we think is valuable and its interpreted which means users can quickly see results without going through a lot of complexity to get started.
